# Security Policy

## Reporting a Vulnerability

I take the security of SimpleARM seriously. If you believe you have found a security vulnerability, please report it to me as described below.

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please report them via email to Mudit Bhargava.

Please include the following information in your report:
- Type of issue (e.g., buffer overflow, logic error, etc.)
- Location of the vulnerability (file and line number if possible)
- Potential impact
- Steps to reproduce

I will acknowledge receipt of your report within 48 hours and provide a timeline for resolution.
